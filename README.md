# PulsePastures: High-Stakes Decentralized RPG 🚜💎

**PulsePastures** is an autonomous, decentralized agricultural yield engine built on the **Somnia Network**. It combines high-fidelity Web3 reactivity with a robust, "High-Stakes" economic model for a professional Play-to-Earn experience.

![PulsePastures Concept](https://raw.githubusercontent.com/lucide-react/lucide/main/icons/pickaxe.svg)

## 🌾 Core Features (Protocol v2.0)

### 🚀 Hardcoded Tiered Economy
- **Livestock Investment**: Tiered entry points from **5 STT** (Chicken) to **80 STT** (Bee).
- **Proportional ROI**: Every harvest cycle is mathematically balanced to provide a consistent **10% Return on Investment** per harvest.
- **Strategic Expansion**: Expanding your farm is a major project milestone, hardcoded at **100 STT** per slot.

### 🛡️ Administrator HUD (Owner Only)
A secure, wallet-locked management console for the project operator:
- **Proportional Withdrawals**: Granular control over treasury retrieval (10%, 25%, 50%, or 100% buttons).
- **Treasury Privacy**: Real-time liquidity monitor (TREASURY LIQ) visible only to the authorized administrator.

### 🎶 Stealth Audio Engine
- **Z-Interaction Playback**: Background music ("Grass Skirt Chase") starts seamlessly upon the first user interaction.
- **MediaSession Suppression**: Advanced browser-level metadata suppression to minimize intrusive media controls.

## 📡 Somnia Reactivity (Advanced)

PulsePastures utilizes Somnia's native reactivity for real-time state synchronization. Our implementation includes:
- **Atomic State Bundling**: Using `ethCalls` to fetch envanter and slot data alongside events in a single block.
- **Push-based Sync**: Eliminates RPC polling, reducing latency and infrastructure costs.
- **Optimization**: `onlyPushChanges` used to minimize frontend updates.

## 🚜 Getting Started

1.  **Connect Wallet**: Link your MetaMask to the Somnia Network.
2.  **Purchase Livestock**: Start with a Chicken (5 STT) or go straight for the Bee (80 STT).
3.  **Harvest**: Collect your yields (Eggs, Meat, Milk, etc.) regularly.
4.  **Liquidate**: Visit the **Merchant Terminal** to sell your products back to the treasury for STT token rewards.
